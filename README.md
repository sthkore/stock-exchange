# stock-exchange
股票交易所 参考 https://liaoxuefeng.com/books/java/springcloud/architecture/index.html

A simple, super fast, 7x24 exchange.

### Technology

- Java 17 + SpringBoot 3.x + Spring 6.x + Maven.
- 100% in-memory trading.
